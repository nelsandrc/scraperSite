Team Covid@Stout

Mission

Thank you for checking out our website. We are a team of seniors in the computer science program at UW-Stout who wanted to create an easy to use and understand platform for people to see the spread of COVID-19.

How It Works

This website was built by four seniors for our capstone project. Using JavaScript code running in an AWS cloud instance, we automatically collect our data from various sources once or more a week. This new data is then sent to our MySQL database which is also running in AWS. The database is queried by a Splunk instance which then generates the visualizations you can see on the main page. Finally, these reports are exported to the website which is running in AWS Amplify. The entire process is automated and will continue to run as long as the code is hosted and the data sources continue publishing new information.
